# /scroll-film

Use this command when you want to build a continuous cinematic animated scroll website (scrollytelling) — where the page plays like one unbroken cinematic shot as the visitor scrolls.

## Flow

1. **Step 0 — Interview**: Asks the key questions about brand, journey, motion type (Lane A pure-code vs Lane B cinematic video footage).
2. **Step 1 — Pitch Concepts**: Develops 2-3 named creative concepts with continuous camera walkthroughs.
3. **Step 2 — Art-Direction & Build**: Sets exact palette, typography, inline SVGs, GSAP/Lenis scrub mechanics or video frame canvas engine.
4. **Step 3 — Verification & Deploy**: Runs jank test verification and optional Vercel deployment.

## Usage

```
/scroll-film VOLTA electric race team continuous camera site
/scroll-film scrollytelling page for luxury perfume brand
```
